# UrbanMart-Retail-Analytics
UrbanMart Retail Analytics — Q1 2024

A full-stack data analytics project analyzing ₹30.72M in retail 
transactions across 5 categories, 5 cities, and 4 customer segments.

What's included:
- 18 SQL queries (Basic → Advanced with Window Functions & LAG)
- Interactive Power BI dashboard with NEXT/BACK navigation
- PowerPoint presentation with SQL insights & recommendations

Key Findings:
- Electronics drives 63% of revenue
- Grocery has highest return rate (9.3%)
- February was peak month at ₹10.99M (+14.6% MoM)
- EMI & Credit Card generate 45% of total profit

Tools: SQL | Power BI | Excel
